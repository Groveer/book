Arch Linux安装后配置

配置源

添加archlinuxcn仓库

  1. 添加文件：/etc/pacman.d/archlinuxcn-mirrorlist
    添加内容：Server = https://repo.archlinuxcn.org/$arch
    更多内容：参考国内仓库
  2. 修改文件：/etc/pacman.conf
    取消注释 #Color
    添加内容：
[archlinuxcn]
Include = /etc/pacman.d/archlinux-mirrorlist
  1. 安装 GPG key：
sudo pacman -Syu
sudo pacman -S archlinuxcn-keyring

若安装失败，可以参考GnuPG-2.1 与 pacman 密钥环重新生成密钥环。

配置aur

sudo pacman -S yay

以后就可以使用yay代替pacman了，语法完全一致，无需提前sudo，并且可访问aur仓库。

安装扩展固件

为了让电脑充分发挥性能，在配置aur仓库后，应安装扩展固件：

yay -S mkinitcpio-firmware

配置控制台登录图形桌面

配置X11环境

  1. 安装xorg：
yay -S xorg-server xorg-xinit
  1. 修改～/.xinitrc文件，并添加内容，挑选自己的桌面环境添加：
exec startplasma-x11       (kde-x11)
exec startdde              (dde)
exec gnome-session         (gnome)
exec openbox-session       (openbox)
exec startxfce4            (xfce)
exec mate-session          (mate)
  1. 修改~/.bash_profile(bash)或者～/.zprofile（zsh）文件，并添加内容：
if [[ -z $DISPLAY && $(tty) == /dev/tty1 ]]; then
  exec startx
fi

配置Wayland环境

wayland本身只是个协议，并不提供图形环境，因此需要安装混合器或直接安装有内置混合器的桌面环境

  1. KDE桌面环境
yay -S plasma-desktop plasma-wayland-session
startplasma-wayland

2、Gnome桌面环境

yay -S gnome-shell
gnome-shell --wayland

其他桌面环境自行搜索：wayland

注意：某些应用程序是支持wayland协议，但是默认却走的xwayland，导致显示效果很不理想，可以设置环境变量或配置文件使其走wayland协议

  1. Firefox： 环境变量：
MOZ_ENABLE_WAYLAND=1
  1. Chromium： 在~/.config/chromium-flags.conf中，添加以下几行:
--enable-features=UseOzonePlatform
--ozone-platform=wayland
--enable-features=WebRTCPipeWireCapturer
  1. qt程序： 需要额外安装：
yay -S qt5-wayland qt6-wayland

然后每次启动程序添加参数：-platform wayland 或直接设置QT_QPA_PLATFORM=wayland，这样就不必每次添加参数

安装oh-my-zsh及插件

  1. 安装必备工具：
yay -S git wget
  1. 安装oh-my-zsh： 创建文件install.sh并写入以下链接中的内容
    修改以下内容：
# Default settings
ZSH=${ZSH:-~/.oh-my-zsh}
REPO=${REPO:-ohmyzsh/ohmyzsh}
REMOTE=${REMOTE:-https://github.com/${REPO}.git}
BRANCH=${BRANCH:-master}

改为：

REPO=${REPO:-mirrors/oh-my-zsh}
REMOTE=${REMOTE:-https://gitee.com/${REPO}.git}

添加可执行权限：

chmod +x install.sh

执行脚本进行安装：

./install.sh
  1. 安装历史记录插件和语法检查插件
cd ~/.oh-my-zsh/plugins
git clone https://github.com/zsh-users/zsh-autosuggestions.git
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git
  1. 修改～/.zshrc文件，找到plugins=(git)，改为：
plugins=(
	git
	sudo
	zsh-syntax-highlighting
	zsh-autosuggestions
)
  1. 使插件生效：
source ~/.zshrc

安装输入法

  1. 安装fcitx5：
yay -S fcitx5 fcitx5-chinese-addons fcitx5-gtk fcitx5-configtool
  1. 配置环境变量/etc/environment：

配置开发环境

python

  1. 下载pip
yay -S python-pip
  1. 设置pip源
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

golang

  1. 安装go：
yay -S go
  1. 选择一个go工作目录，以～/Develop/go为例，新建三个目录：
mkdir -p ~/Develop/go/{src,bin,pkg}
  1. 配置环境变量~/.bash_profile，zsh环境为~/.zprofile：
export GOROOT=/usr/lib/go
export GOPATH=~/Develop/go		   # 这两行你需要
export GOBIN=~/Develop/go/bin		# 修改为自己的
export PATH=$PATH:$GOROOT/bin:$GOBIN

source ~/bash_profile
  1. 配置GOPROXY
go env -w GOPROXY=https://goproxy.io,direct

安装KDE桌面环境

  1. 最小化安装：
yay -S plasma-desktop
yay -S plasma-nm plasma-pa kscreen bluedevil powerdevil kwalletmanager konsole

plasma-desktop：基本的KDE桌面环境
plasma-nm：网络管理模块，右下角的网络连接以及系统设置中的网络设置
plasma-pa：声音模块，右下角的声音设置以及系统设置中的音频设置
kscreen：屏幕管理模块，系统设置中的显示器配置
bluedevil：蓝牙模块，启用需systemctl enable bluetooth
powerdevil：电源管理模块，系统设置中的电源管理
kwalletmanager：KDE 钱包管理，一般用来禁用电子钱包
konsole：KDE 的仿真终端
2. 基本安装：
基本安装并不代表最小化安装，基本安装会附带很多程序，如桌面小工具、discover等

yay -S plasma
  1. 安装KDE应用： 注意，这仅仅安装应用程序，并不会安装 Plasma 桌面。
yay -S kde-applications
